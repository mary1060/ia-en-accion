# IA en Acción — Newsletter

Newsletter semanal de inteligencia artificial aplicada por **Mary Cárdenas de Gil** · Ateneo Empresarial.

🌐 **URL:** [newsletter.marycardenas.ai](https://newsletter.marycardenas.ai)

## Estructura del repositorio

```
ia-en-accion/
├── index.html          ← Página de inicio (archivo de ediciones)
├── vercel.json         ← Configuración de Vercel
├── README.md
└── edicion-12/
│   └── index.html      ← Edición #12 (14–20 Abril 2026)
└── edicion-13/         ← (próxima edición)
    └── index.html
```

## Cómo agregar una nueva edición

1. Crear carpeta `edicion-XX/`
2. Guardar el HTML del newsletter como `edicion-XX/index.html`
3. Agregar la tarjeta de esa edición en `index.html` (sección "Ediciones publicadas")
4. Hacer commit y push — Vercel despliega automáticamente
